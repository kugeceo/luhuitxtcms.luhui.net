常见问题：

 

什么是网站主？

是广告交易双方的其中一方，即网站的拥有者，具有修改、新增、删除网站内容的权力，并承担相关法律责任的法人。在自已网站上投放广告主的广告后，并按照本平台规定通过本平台收取佣金。

 

成为网站主的先决条件是什么？

1. 网站主的网站必须拥有自己的国际顶级域名或国家顶级域名。 
2. 网站世界排名原则上必须在20万位以内。 
3. 网站中不得包含违法及不健康内容，否则一律取消本站会员资格，且引起的一切后果自负，并交由相关部门处理（我们定期会对网站主进行资格审查）。

 

网站主注册会员和使用本系统是否要付费？

完全免费！鲁虺网店推广联盟所提供的所有“网站主”会员功能均为免费使用。如果网站主在非本系统业务范围内有特殊要求，我们才会对其收取相关费用。

 

网站主会员审核标准是什么？

1. 网站本身及广告不能包含任何违反国家法律的内容。 
2. 网站本身及广告不能含有恶性代码及病毒，不能包含不健康的内容，如成人、性、色情、淫秽、赌博、暴力、反动等等。 
3. 必须有自己的国际顶级域名或国家顶级域名(如:http://www.abc.com)，原则上不通过使用二级域名或免费域名网站的审核。 
4. 网站世界排名在20万以内，以周排名和三个月平均排名数据为准。排名查询请到:http://www.alexa.com 
5. 对只有论坛或无实际内容或页面排版不够专业美观的网站一律不予通过。 
6. 每个网页的弹出窗口不得超过1个，不得自动弹出一次以上要求设为首页或加入收藏等类似对话框。 
补充说明：如网站在通过审核后违反或不再符合以上标准中的任何一条，鲁虺网店推广联盟将保留与其终止合作关系的权力。

 

怎样成为网站主会员？
1. 正确填写注册表格，注册成为鲁虺网店推广联盟待审会员。 
2. 鲁虺网店推广联盟将对于申请者的个人资料和网站进行资质审核，对通过审核的网站主会员尽快发出确认通知。 
3. 成为正式网站主会员后可登陆会员管理界面获得广告代码。

 

网站主会员的权利有哪些？

1. 会员可获得完全免费的技术咨询和技术支持 
2. 网站主会员可完全免费使用本系统(特殊要求除外) 
3. 可以免费获得大量经过资质审核的广告资源及广告投放权 
4. 可实时根据广告主发布的广告信息，完全自主地选择投放某广告 
5. 可通过实时的报表系统准确直观地了解自己正在投放的各广告情况，以及由此带来的收入
6. 通过鲁虺网店推广联盟对广告主的资质审核及预付款制度，网站主能保障在广告投放结束后立即获得广告费等权益 
7. 可以准时享受鲁虺网店推广联盟完善的佣金结算服务，每周定期收到自己应得的广告收入 8. 拥有完全公开的交易信息，竞争环境绝对公正公平

 

网站主会员的义务有哪些？

1. 网站主必须保证其网站的所有网页至始至终不包含任何注册协议中定义的违规内容，并且保证不链接到含有违规内容的网页。 
2. 如果网站主提供给鲁虺网店推广联盟平台的资料、信息等发生变化，应及时登陆鲁虺网店推广联盟平台进行修改与确认。 
3. 鲁虺网店推广联盟有权随时单方面决定更改本服务条款。如服务条款有任何变更，鲁虺网店推广联盟科技将在“鲁虺网店推广联盟平台”上刊载公告通知网站主，且经修订的服务条款自公布之时开始生效。 本服务条款变更后，如网站主继续使用鲁虺网店推广联盟提供的服务，则表示网站主已接受经修订的服务条款。

 

鲁虺网店推广联盟如何对网站主计费？

鲁虺网店推广联盟平台将按以下形式计费：
1.实际有效点击次数
2.实际产生的注册量或销售额

 

鲁虺网店推广联盟对网站主的付费方式有哪些？

目前支持支付宝、工商银行转帐、招商银行、建设银行转帐方式，如果没有以上两种卡，请到所在地区办理（在您的支付方式中务必填写银行卡帐号和您的户名以及详细的开户行地址）。

 

鲁虺网店推广联盟有拖欠网站主广告佣金的可能性吗？

鲁虺网店推广联盟作为服务第三方旨在为广告主及网站主提供公平、公正的交易平台，而我们对广告主采取的资质审查与预付款制度，可以保证绝无拖欠网站主佣金的可能性。

 

网站主会员在什么情况下会被拒付广告费？

1、网站本身及广告违反了国家法律或含有恶性代码与病毒，及包含成人、性、色情、淫秽、赌博、暴力、反动等等不健康的内容。 
2、所投放广告的网页不属于会员拥有。 
3、牵涉到知识产权纠纷的网站（如非法 MP3 、盗版软件网站，黑客站点，软件序列号站点，注册机、注册码站点，或链接至这些网页的站点）。 
4、链接、讨论或提供影响网络秩序的内容，如提供邮件炸弹、计算机病毒等的网站。 
5、被查出更改标准 HTML 代码的网站。 
6、被查出同一人注册多个帐号的网站。 
7、使用非 HTML 手段、 JAVASRIPT 窗口、隐藏 FRAME 、CGI 自动生成、网页 REFRESH/AUTOLOAD 等手段来演绎广告代码。 
8、不可由会员本人或指示、暗示第三方点击广告以获取非法广告费。

 

什么情况下网站主才会获得广告佣金？

网站主在自己的网站上投放自己选择的广告，如果其网站的访客发生与产生了该广告活动中注明的行为与效果，则网站主即可相应得到该广告活动中注明的收入。

 

如何获取广告代码并让广告显示在网站上？

第一步：注册成为鲁虺网店推广联盟正式网站主会员。 
第二步：登陆系统选择广告代码，按广告类型选择代码

第三步：将选择的广告代码复制到您的页面中，整个广告投放的操作到此完毕。

 

关于代码的介绍和使用

在会员区的广告代码栏目里有多种形式的广告代码，网站主会员可根据各广告的尺寸和价格来选择最合适的代码放置到页面上。每一种类型的广告都会分别独立统计计费。

 

是不是所有的点击次数都有记录呢？

并不是所有的点击次数都有记录，因为我们计算一次有效点击是按照同一访问者(同一IP)点击一次广告记录为标准，为防止作弊，同一访问者在24小时内多次点击不作统计。 有的会员为了试验广告是否准确计费，不断点击或刷新，但同一 IP 的连续点击和刷新是不计算的，所以建议您把页面上传到服务器上让访问者点击来察看检验结果。

 

广告点击率有限制吗？

广告点击率 = 广告被点击的次数(Clicks) / 广告被显示的次数(Opens) 
广告点弹率是有上限的，上限值为系统设定的最高点/显比例。如果点显率超过此上限值，点击依然记录，但不再计费。限值是经过长期验证并适当放宽比例后设定，只要你的广告以正常形式投入，肯定不会达到限值。

 

可否把广告代码放在多个页面里？或者同一个页面放多个广告代码？

为了增加点击的机率，或是避免未知因素导致某页面 无法弹出窗口或无法点击，您可以在多个页面上投放广告代码或在同一个页面投放多个广告代码。

 

网站主作弊行为判断标准？

1. 任何人工产生的弹窗，代理服务器 IP 弹窗，假 IP 弹窗，相互弹窗，自动刷新， javascript都是作弊。使用非HTML手段、JAVASRIPT 窗口、隐藏FRAME 、CGI 自动生成、网页REFRESH/AUTOLOAD 等手段来演绎广告代码均属无效弹窗。 


2. 来自其他站点的弹窗。比如：域名为http://www.abc.com的站点 A 在鲁虺网店推广联盟注册并通过审核，那么属于A 站点的广告代码不能放在除 A 站点外的其他任何网站上，鲁虺网店推广联盟也不承认来自其他站点的弹窗，并视其为作弊行为。 


3. 不得以任何语言或者形式诱导、误导、要求或强制访问者点击广告。

　　 

作弊范例A：支持本站发展，请点击广告。（注释：如网页出现以上文字即判定为作弊。如出现“点击下面广告……”的文字，则该文字的水平位置以下的任意位置都不能出现鲁虺网店推广联盟投放的广告。其他具有方向性的文字同理。） 　　 

作弊范例B：点击广告获取下载地址 / 观看影片 / 提高下载速度 / 注册帐号。（注释：此手段为条件交换，情节严重。出现方向性文字如：点击下面广告获取，则判定方法如作弊范例 1 中注释所述。）


4. 广告条必须完整嵌入网页且完全显示，不得删减、修改代码，不得将鲁虺网店推广联盟的广告放在仅有广告而无其他实际内容的页面（为方便修改广告投放而用 iframe 或 javascript 方法嵌入网页的不在此范围）。不得使用误导访客点击的手段。

　　 

误导范例A：网站主自行使用弹出窗口/层，误导访客点击。 　　 

误导范例 B：广告图片与其周围的网页元素（文字或图片连接）的距离过近，促使访客误点！要求：网页元素和广告图片的距离要多于 15 象素，以保证访客不会误点。 

　　 

鲁虺网店推广联盟允许的方式： 
1. 正常投放广告，没有使用任何手段的都是鲁虺网店推广联盟允许并推荐的方式。正常投放广告的网站主完全不必担心会被判定为作弊。 
2. 鲁虺网店推广联盟规定网站首页必须投放一个广告(即在首页页中插入从鲁虺网店推广联盟获取的任一广告代码)。 
3. 允许提示的文字：可增强广告效果、不会给广告主造成负面影响的提示文字。 
范例A：本站推荐最好的广告联盟。 
范例B：××××上市，点击下面广告了解详细信息。


注： 以上为鲁虺网店推广联盟判定作弊网站的标准并由发布日起严格执行。 鲁虺网店推广联盟平台会记录每次弹窗和点击在案。如发现以上作弊行为，将清空其当月佣金记录，情节严重者删除 ID 并永不通过审核。 鲁虺网店推广联盟有权利对以上判定网站主作弊的标准根据实际情况作出相应的修改、扩充和完善。鲁虺网店推广联盟保留最终解释权。

 

广告统计数据多长时间更新一次（网站主）？

弹窗和点击数据为实时统计更新。即网站主会员可以实时看到时事的广告数据，销售型广告根据广告主，一般为每天汇总前一天数据理，可以查看汇总后的历史数据统计、佣金金额、历史佣金统计及历史支付记录。

 

是否支持网站主发展下线？

支持发展下线。 
网站主会员将鲁虺网店推广联盟网络广告平台介绍给其他网站主，且其下线与鲁虺网店推广联盟平台发生合作关系即发展下线，则鲁虺网店推广联盟平台将下线网站主第一个月广告佣金的10％奖励给发展下线的网站主。网站主可以在“下线佣金查询”里面查看到下线网站主的广告数据及金额。

每当联盟成员提供给鲁虺网店推广联盟个人信息时，鲁虺网店推广联盟会采取合理的步骤保护联盟成员的个人信息，鲁虺网店推广联盟也会采取合理的安全手段保护已存储的个人信息。由于网上技术发展的突飞猛进，鲁虺网店推广联盟会随时更新并公布鲁虺网店推广联盟的信息保密制度，除非以下三种情况外，鲁虺网店推广联盟不会在未经合法联盟成员授权时，公开、编辑或透露联盟成员资料及其它保存在鲁虺网店推广联盟中的保密性内容：

2) 有关权力机关要求；
3)联盟成员授权；

（包括，如果联盟成员要求鲁虺网店推广联盟提供特定服务时，鲁虺网店推广联盟则需要把联盟成员的姓名和地址提供给与此相关联的第三方如邮政服务单位）。

六、 风险分担
联盟成员使用本网站将承担一定风险：鲁虺网店推广联盟将不承担由于联盟成员自身过错、技术或其它不可控原因导致网站页面信息或其它方面的错误而给联盟成员造成的损失。
